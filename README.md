# 離乳食レシピアプリ

離乳食のレシピを提案・保存するスマートフォン向けアプリです。冷蔵庫の在庫管理機能を持ち、栄養バランスの取れた離乳食レシピを自動生成します。

## 特徴

- 🍚 **レシピ自動生成**: 朝・昼・夜の食事タイプに応じて、栄養バランスの取れたレシピを提案
- 🥕 **在庫管理**: 冷蔵庫にある食材を管理し、在庫から最適なレシピを生成
- 📅 **履歴管理**: 過去3日分のレシピ履歴を確認可能
- 📱 **PWA対応**: ホーム画面に追加してアプリのように使用可能
- 🔒 **完全ローカル**: クラウドサーバ不要、すべてのデータはデバイスに保存
- ⚡ **軽量**: 外部ライブラリ不使用、高速起動

## インストール方法

### スマートフォンでの使用

1. **ファイルの準備**
   - `baby-food-recipe-app.zip` を解凍します

2. **ブラウザで開く**
   - 解凍したフォルダ内の `index.html` をスマートフォンのブラウザで開きます
   - または、ファイルをWebサーバーにアップロードしてアクセスします

3. **ホーム画面に追加**（推奨）

   **iOS (Safari):**
   - 画面下部の共有ボタン（□に↑）をタップ
   - 「ホーム画面に追加」を選択
   - 「追加」をタップ

   **Android (Chrome):**
   - 画面右上のメニュー（⋮）をタップ
   - 「ホーム画面に追加」を選択
   - 「追加」をタップ

4. **アプリとして起動**
   - ホーム画面のアイコンからアプリを起動
   - 全画面表示で、ネイティブアプリのように使用できます

### PCでの使用

1. `index.html` をブラウザで開くだけで使用できます
2. ローカルサーバーを起動する場合:
   ```bash
   # Python 3の場合
   python -m http.server 8000
   
   # Node.jsの場合
   npx http-server
   ```
3. ブラウザで `http://localhost:8000` にアクセス

## 使い方

### 1. 在庫管理

1. 「在庫管理」タブを開く
2. 冷蔵庫にある食材をタップして在庫ありに設定（✓マーク）
3. カスタム食材を追加する場合は「➕ カスタム食材追加」ボタンをクリック

### 2. レシピ生成

1. 「レシピ生成」タブを開く
2. 朝食・昼食・夕食のいずれかを選択
3. 在庫から自動的に栄養バランスの取れたレシピが生成されます

**レシピの構成:**
- 炭水化物 × 1
- タンパク質 × 1
- 野菜 × 3
- 乳製品 × 0-1（在庫がある場合）
- フルーツ × 0-1（朝食のみ）

### 3. 履歴確認

1. 「履歴」タブを開く
2. 過去3日分のレシピ履歴を確認できます
3. 日付ごとに朝・昼・夜のレシピが表示されます

## トラブルシューティング

### レシピが生成されない

- 在庫管理で必要な食材カテゴリ（炭水化物、タンパク質、野菜）に在庫があるか確認してください
- 最低でも各カテゴリに1つ以上の食材が必要です

### データが保存されない

- ブラウザのプライベートモード（シークレットモード）では使用できません
- ブラウザの設定でLocalStorageが有効になっているか確認してください

### アプリが起動しない

- ブラウザのキャッシュをクリアして再度試してください
- 最新のブラウザバージョンを使用してください
