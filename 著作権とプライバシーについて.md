# 著作権とプライバシーについて

## 著作権について

### あなたが作成したアプリの著作権

✅ **著作権はあなたに帰属します**

GitHubにアップロードしても、著作権はあなたのものです。
GitHubは単なるホスティングサービスであり、著作権を主張しません。

### GitHubの利用規約

GitHub利用規約では：
- あなたのコンテンツの著作権はあなたに残る
- GitHubはホスティングのために必要な権利のみを持つ
- あなたはいつでもコンテンツを削除できる

参考：https://docs.github.com/ja/site-policy/github-terms/github-terms-of-service

---

## Publicリポジトリの意味

### 公開される情報

❌ **公開される**
- アプリのソースコード（HTML、CSS、JavaScript）
- アイコン画像
- README などのドキュメント

✅ **公開されない**
- あなたの在庫データ
- あなたのレシピ履歴
- 個人情報

**理由：** データはスマホのLocalStorageに保存されるため、
GitHubには一切アップロードされません。

## ライセンスの選択

### オプション1: ライセンスなし（デフォルト）

ライセンスファイルを追加しない場合：
- 著作権はあなたに帰属
- 他の人は閲覧できるが、使用・改変・再配布はできない
- 最も制限的

### オプション2: MITライセンス（推奨）

自由に使ってもらいたい場合：
- 他の人も自由に使用・改変・再配布できる
- 著作権表示は必要
- 無保証（責任を負わない）

### オプション3: プライベートリポジトリ

**完全に非公開にしたい場合：**

GitHubの無料アカウントでもプライベートリポジトリが使えます。

**ただし、GitHub Pagesの制限：**
- 無料プランではプライベートリポジトリのGitHub Pagesは使えない
- GitHub Pro（有料）が必要

**メリット：**
- 完全無料
- 設定が簡単
- 他の人の役に立つ

## プライバシーについて

### あなたのデータは安全

✅ **在庫データ**
- スマホのLocalStorageに保存
- インターネットに送信されない
- GitHubに保存されない

✅ **レシピ履歴**
- スマホのLocalStorageに保存
- インターネットに送信されない
- GitHubに保存されない

✅ **個人情報**
- アプリは個人情報を収集しない
- 名前、メールアドレスなどは不要

### データの保存場所

```
[あなたのスマホ]
  └─ ブラウザのLocalStorage
      ├─ 在庫データ
      └─ レシピ履歴

[GitHub]
  └─ アプリのファイルのみ
      ├─ HTML
      ├─ CSS
      └─ JavaScript
```

**データは完全に分離されています！**

## 代替案：完全プライベートなホスティング

### Netlify（推奨）

**メリット：**
- ✅ 完全無料
- ✅ プライベート可能
- ✅ ドラッグ&ドロップでアップロード
- ✅ HTTPSで安全
- ✅ 高速

**手順：**
1. https://www.netlify.com でアカウント作成
2. 「Add new site」→「Deploy manually」
3. フォルダをドラッグ&ドロップ
4. 発行されたURLにアクセス

**プライバシー：**
- ソースコードも非公開にできる
- URLを知っている人だけがアクセス可能

### Vercel

**メリット：**
- ✅ 完全無料
- ✅ プライベート可能
- ✅ 高速
- ✅ HTTPSで安全

**手順：**
1. https://vercel.com でアカウント作成
2. 「Add New Project」
3. フォルダをアップロード
4. 発行されたURLにアクセス

## よくある質問

**Q: GitHubに個人情報が漏れる？**
A: いいえ。アプリのコードだけがアップロードされ、
   あなたのデータはスマホに保存されます。

**Q: 他の人が私のレシピを見られる？**
A: いいえ。レシピはあなたのスマホにのみ保存されます。

**Q: ソースコードを公開したくない**
A: NetlifyまたはVercelを使用してください。
   完全にプライベートにできます。

**Q: 著作権を主張したい**
A: リポジトリにLICENSEファイルを追加し、
   「Copyright (c) 2025 [あなたの名前]」と記載してください。

**Q: 商用利用できる？**
A: はい。このアプリは商用利用も可能です。
   ただし、無保証です。
